### Prerequisites

**Node version 18.7.x**

### Cloning the repository

```shell
git clone https://github.com/Screetch/next-trpc-prisma.git
```

### Install packages

```shell
yarn install
```

### Setup .env file

```js
DATABASE_URL=
```

### Setup Prisma

```shell
npx prisma generate
npx prisma db push
```

### Start the app

```shell
yarn run dev
```

## Available commands

Running commands with yarn `yarn run [command]`

| command | description                              |
| :------ | :--------------------------------------- |
| `dev`   | Starts a development instance of the app |
